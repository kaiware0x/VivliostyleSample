An academic theme for Vivliostyle.

## Install

```bash
npm install --save @vivliostyle/theme-academic
# or
yarn add @vivliostyle/theme-academic
```

## Use

In `vivliostyle.config.js`:

```js
module.exports = {
  theme: '@vivliostyle/theme-academic',
};
```
